# sber

Задачи на Java:

1. Напишите Java-программу, которая:
   a. Принимает на вход: логин, пароль и имя хоста БД;
   b. Осуществляет подключение к БД (мы точно знаем, что это Postgresql);
   c. Считает количество записей в таблице dq_rule;
   d. Создаёт строку в таблице log – в таблице два поля: дата/время создания строки и результат пункта c;

2. Напишите пару Java-классов. Родительский и дочерний.
   a. В родительском классе должен быть метод, который возвращает факториал любого числа, переданного как параметр метода;
   b. Дочерний переопределяет этот метод и возвращает всегда факториал 10и;
   c. Сделайте так, чтобы нельзя было создать объект от родительского класса – только от дочернего;

3. Добавьте к решению предыдущей задачи третий класс – такой, который позволит получать факториал любого числа без создания экземпляра класса (объекта);

4. Напишите Java-программу, которая принимает на вход текстовый файл с параметрами в форме «ключ-значение»;
   a. Программа должна прочитать файл и выдать его содержимое в отсортированном по ключу виде;

5. Напишите Java-программу, которая читает текстовый файл, содержащий логин и пароль;

   a. Шифрует пароль с использованием ключа, который также вшифровывается в результат шифрования;

   b. Создаёт новый файл с шифрованным паролем и удаляет старый файл с открытым паролем;

   c. Другой метод того же класса должен уметь расшифровать таким образом зашифрованный пароль из файла с шифрованным паролем и печатать расшифрованный пароль в консоль, исходный файл с расшифрованным паролем создавать не надо;

   d. Без внешне передаваемого ключа шифрования. Ключ шифрования должен генерироваться самой программой и становится частью шифра, но только так, чтобы его нельзя было легко вычленить.
